# TUF-rit.github.io

这是一个 GitHub Pages 网站，托管在 `https://tuf-rit.github.io/`。

## 项目说明

本项目包含一个现代化的网页应用，展示系统诊断报告界面。

### 功能特性

- **现代设计**：采用玻璃拟态（Glassmorphism）设计风格
- **响应式布局**：完美适配各种屏幕尺寸
- **无障碍访问**：支持减少动画偏好，符合 WCAG 标准
- **流畅动画**：脉冲效果和浮动动画增强视觉体验

### 文件结构

```
.
├── index.html          # 主页面（GitHub Pages 入口）
├── pdf/
│   └── index.html      # 原始文件备份
└── README.md           # 项目说明文档
```

### 访问方式

- **主页**：https://tuf-rit.github.io/
- **备用路径**：https://tuf-rit.github.io/pdf/

### 技术栈

- HTML5
- CSS3（包括 Backdrop Filter、CSS Grid、Flexbox）
- 原生 JavaScript（无依赖）

### 浏览器兼容性

支持所有现代浏览器，包括：
- Chrome/Edge 90+
- Firefox 88+
- Safari 15+

### 部署信息

本项目通过 GitHub Pages 自动部署，每次推送到 `main` 分支后会自动更新。
